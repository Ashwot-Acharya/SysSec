# Full-Stack Mini Web Application: File Management System

This is a student project demonstrating a full-stack file management system using React (Vite) and Node.js (Express).

## Features
- **Authentication**: JWT, bcrypt, localStorage.
- **File Uploads**: Uses Multer, stored on the local server.
- **CRUD Files**: View list, upload, rename, and delete files.
- **Database**: Uses a simple local `database.json` for zero-setup execution.

## Getting Started

### 1. Run the Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Run the Frontend
In a new terminal window:
```bash
cd frontend
npm install
npm run dev
```

### 3. Usage
- Go to the frontend URL (usually `http://localhost:5173`).
- Create an account using the Register page.
- Login and start uploading files!
